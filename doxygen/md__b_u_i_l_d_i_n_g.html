Development setup

Visual Studio Code

  1. Open the folder in Visual Studio Code.
  2. Install the recommended extensions.
    (When opening the folder, there will be a notification that asks whether to install the recommended extensions.)
    • CMake Tools
    • C/C++
  3. Platform-specific setup
    • Windows MSVC
      1. Install Visual Studio Build Tools (Windows SDK and MSVC - VS 2022).
      2. Install Qt 5.15 including MSVC 2019.
      3. Set the environment variable QT_DIR to <Qt Install dir>/5.15.2/msvc2019_64.
        This can be done by adding ```json "cmake.environment": { "QT_DIR": "<Qt Dir>/5.15.2/msvc2019_64" }, ``` to the vscode settings, or adding that variable globally in Windows.
