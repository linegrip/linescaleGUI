Detailed Description

This file contains the class to control all connected devices.

Authors
Gschwind, Weber, Schoch, Niederberger

The CommMaster handles all communication with the connected devices. It abstracts the different types of connections so from this stage upwards the type of connection doesn't matter anymore.

Each device gets a label which is used as an identifier inside the entire code: USB: Port name BLE: tbd
